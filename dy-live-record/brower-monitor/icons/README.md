# 图标文件说明

请在此文件夹中放置以下尺寸的图标文件：

- **icon16.png** - 16x16 像素
- **icon32.png** - 32x32 像素
- **icon48.png** - 48x48 像素
- **icon128.png** - 128x128 像素

## 图标设计建议

1. 使用透明背景的PNG格式
2. 图标应该简洁明了，能够在小尺寸下清晰显示
3. 建议使用与监控相关的图标，如：
   - 🔍 放大镜
   - 📡 信号/监控
   - 🌐 地球/网络
   - 👁️ 眼睛
   
## 在线图标生成工具

如果您需要快速生成图标，可以使用以下在线工具：

1. [Favicon Generator](https://favicon.io/)
2. [RealFaviconGenerator](https://realfavicongenerator.net/)
3. [IconKitchen](https://icon.kitchen/)

## 临时解决方案

如果暂时没有图标，可以使用任何PNG图片，浏览器会自动缩放。但为了最佳显示效果，建议准备各个尺寸的图标。
